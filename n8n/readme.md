## Environment Variables
- `N8N_HOST`: Hostname for n8n (e.g., `automation.example.com`)
- `N8N_PORT`: Port to expose n8n on the host (e.g., `5678`)
- `N8N_PROTOCOL`: Protocol for n8n (e.g., `https`)
- `GENERIC_TIMEZONE`: Timezone for the container (e.g., `Europe/Berlin`)
- `N8N_CONCURRENCY_PRODUCTION_LIMIT`: Maximum number of concurrent workflow executions
